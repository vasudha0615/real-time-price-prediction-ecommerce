# Real-Time Price Prediction System

## Project Overview
This project demonstrates a real-time price prediction pipeline using
Apache Kafka, Machine Learning, Snowflake, and Power BI.

## Architecture
Kafka Producer → ML Predictor → CSV → Snowflake → Power BI

## Tech Stack
- Python
- Apache Kafka
- Scikit-learn
- Snowflake
- Power BI

## Key Features
- Real-time data streaming using Kafka
- Machine learning-based price prediction
- Cloud data storage in Snowflake
- Business dashboards using Power BI

## Status
Working end-to-end pipeline with real-time predictions.
