# Machine Learning Model

## Model Used
RandomForest Regressor

## Input Features
- CustomerID
- Quantity
- Year
- Month

## Target
Predicted product price based on historical retail data.

## Training
Model was trained using historical e-commerce transaction data.
Date features were extracted from invoice timestamps.

## Note
The trained model file (`model.pkl`) is not pushed to GitHub.
It is generated locally during training.
